# PRD.md — OTD 助手产品需求文档

## 1. 产品概述

### 1.1 产品定位

OTD（Order to Delivery）助手是一款面向企业内部用户的 SAP ERP 数据查询系统，提供 Web 端和微信小程序端的数据查询能力。系统对接 SAP S/4HANA，覆盖销售订单、生产订单、库存、交货单、开票单据等 8 大核心业务模块，帮助业务人员快速检索和追踪订单全生命周期数据。

### 1.2 目标用户

| 角色 | 典型场景 | 使用频率 |
|------|---------|---------|
| 销售经理 | 查询销售订单状态、客户信息 | 每日 |
| 生产计划员 | 跟踪生产订单进度、实际产出 | 每日 |
| 仓库管理员 | 查询库存、交货单、物料凭证 | 每日 |
| 财务人员 | 核对开票单据 | 每周 |
| 管理层 | 查看 KPI 概览 | 每周 |

### 1.3 核心价值

- **统一入口**：一个平台查询所有 SAP 业务数据，无需登录 SAP GUI
- **移动办公**：微信小程序 + 响应式 Web，随时随地查询
- **关联追踪**：从销售订单一键穿透到发货、开票、生产订单
- **快速上手**：Fiori 3 设计语言，企业用户零学习成本

## 2. 功能需求

### 2.1 用户认证 (P0)

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|---------|
| AUTH-01 | 用户注册 | 用户名 + 密码 + 可选邮箱/显示名 | 注册成功后自动登录 |
| AUTH-02 | 用户登录 | 用户名 + 密码 + 4位数字验证码 | 校验通过签发 JWT，失败后提示重新输入 |
| AUTH-03 | 验证码 | 动态生成4位数字SVG图片 | 点击刷新，5分钟过期 |
| AUTH-04 | 会话管理 | JWT Token (httpOnly Cookie) | 7天有效期，退出清除 |
| AUTH-05 | 路由保护 | 未登录重定向到登录页 | 白名单外全部拦截 |

### 2.2 首页仪表盘 (P1)

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|---------|
| HOME-01 | KPI 卡片 | 销售订单数/生产订单数/库存SKU/客户数 | 通过 `/api/dashboard` 实时聚合 SAP 数据，带趋势 |
| HOME-02 | 功能磁贴 | 8大模块入口 + 图标 + 描述 | 点击跳转对应列表 |

## 3. 非功能需求

### 3.1 性能

| 指标 | 目标 |
|------|------|
| 首屏加载 (LCP) | < 2s |
| API 响应时间 | < 3s (SAP代理) |
| 列表加载 | 20条/页，增量加载 |

### 3.2 兼容性

| 平台 | 支持 |
|------|------|
| Chrome | 最新 2 个版本 |
| Safari | 最新 2 个版本 |
| Edge | 最新 2 个版本 |
| 微信小程序 | 最新基础库（已覆盖 iOS/Android 真机） |

### 3.3 响应式

| 断点 | 宽度 | 行为 |
|------|------|------|
| Mobile | <640px | 卡片视图，侧边栏 overlay |
| Tablet | 640-1024px | 可切换，侧边栏折叠 |
| Desktop | ≥1024px | 表格视图，侧边栏展开 |

### 3.4 安全

| 要求 | 实现 |
|------|------|
| 密码存储 | bcryptjs 哈希 |
| 会话管理 | JWT + httpOnly Cookie |
| SAP 凭证 | 服务端环境变量，不暴露前端 |
| 路由保护 | 中间件拦截未认证请求 |
| 验证码 | 每次登录需输入动态验证码，降低暴力破解风险 |

## 4. 用户故事

### US-01: 销售经理查看订单状态
> 作为销售经理，我希望在手机上快速查看某个销售订单的发货和开票状态，以便回复客户询问。

**流程**：打开小程序 → 登录 → 销售订单列表 → 点击订单 → 点击行项目 → 查看关联发货单和开票单

### US-02: 生产计划员跟踪生产进度
> 作为生产计划员，我希望查看生产订单的实际产出数量，以便判断是否需要调整排产。

**流程**：打开 Web → 登录 → 生产订单列表 → 点击订单 → 查看计划数量 vs 实际产出

### US-03: 仓库管理员查询库存
> 作为仓库管理员，我希望快速查询某个物料的当前库存，以便安排出库。

**流程**：打开 Web → 登录 → 库存查询 → 搜索物料 → 查看各工厂库存

### US-04: 财务人员核对开票
> 作为财务人员，我希望查看开票单据的日期和金额，以便与销售订单核对。

**流程**：打开 Web → 登录 → 开票单据列表 → 查看开票日期和金额

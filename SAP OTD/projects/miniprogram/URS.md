# OTD助手 微信小程序 - 用户需求规格说明书 (URS)

## 1. 概述

### 1.1 项目背景
OTD助手 Web 端已具备完整的 SAP ERP 数据查询能力，包括 8 大业务实体查询、Dashboard 驾驶舱、全局搜索、登录认证、Excel 导出、详情页（销售订单/生产订单）等功能。微信小程序端目前仅有基础列表查询，功能差距较大，需补齐。

### 1.2 目标用户
企业内部员工（生产、销售、仓储、财务等岗位），通过微信小程序随时随地查询 SAP 业务数据。

### 1.3 当前状态
| 功能 | Web 端 | 小程序端 |
|------|--------|----------|
| 8 大实体列表查询 | ✅ | ✅ |
| 登录/注册 | ✅ | ❌ |
| Dashboard 驾驶舱 | ✅ | ❌ |
| 全局搜索 (Cmd+K) | ✅ | ❌ |
| 销售订单详情页 | ✅ (KPI+行项目+定价+发货+开票+生产订单) | ❌ (仅弹窗提示) |
| 生产订单详情页 | ✅ (KPI+组件+工序) | ❌ (仅弹窗提示) |
| 交货单详情页 | ❌ | ❌ |
| 开票单据详情页 | ❌ | ❌ |
| 视图切换 (表格/卡片) | ✅ | ❌ |
| Excel 导出 | ✅ | ❌ |

## 2. 功能需求

### 2.1 登录认证 (P0)

**需求描述**：小程序启动时检查登录态，未登录跳转登录页。

**功能点**：
- 登录页：用户名 + 密码 + 验证码（4位数字）
- 注册页：用户名 + 密码 + 确认密码 + 验证码
- 登录态持久化：使用 `wx.setStorageSync` 存储 token
- 所有 API 请求自动携带 `x-session` header
- 登录过期自动跳转登录页

**接口**：复用 Web 端 `/api/auth/*` 系列接口

---

### 2.2 首页 Dashboard (P0)

**需求描述**：首页从静态模块入口升级为数据驾驶舱。

**功能点**：
- 顶部 KPI 卡片：销售订单数、生产订单数、待发货数、库存条目数
- 最近活动流（最近完成的生产订单等）
- 8 大模块快捷入口（保留现有网格布局，缩小为图标+文字）
- 下拉刷新更新数据

**接口**：复用 Web 端 `/api/dashboard`

---

### 2.3 全局搜索 (P1)

**需求描述**：首页顶部搜索栏，输入关键词跨 8 大实体搜索。

**功能点**：
- 搜索框实时搜索（300ms 防抖）
- 结果按实体类型分组显示
- 点击结果跳转对应详情页
- 搜索历史（本地存储，最多 10 条）

**接口**：复用 Web 端 `/api/search?q=xxx`

---

### 2.4 销售订单详情页 (P0)

**需求描述**：从列表点击销售订单进入完整详情页。

**功能点**：
- **头部信息**：订单号、类型、客户、日期、金额、状态
- **KPI 卡片**：订单金额、行项目数、交货进度、开票状态
- **行项目列表**：物料、描述、数量、单价、净额、交货数量、状态
- **定价明细**：条件类型、费率、金额（可折叠）
- **发货信息**：关联交货单列表（单号、数量、日期、状态）
- **开票信息**：关联开票单据列表（单号、金额、日期、状态）
- **生产订单**：MTO 订单关联的生产订单列表
- **业务伙伴**：售达方/送达方/收款方/付款方地址信息

**接口**：`/api/sap/CE_SALESORDER_0001/SalesOrder?id=xxx&expand=_Item,_Partner,_PricingElement` + `/api/sap/sales-order/:id/related`

---

### 2.5 生产订单详情页 (P0)

**需求描述**：从列表点击生产订单进入完整详情页。

**功能点**：
- **头部信息**：订单号、类型、产品、工厂、状态
- **KPI 卡片**：计划数量、实际交付、组件数、工序数
- **工序进度表**：工序号、工作中心、状态、计划/实际日期、进度条
- **组件清单**：物料、需求数量、已提货数量、齐套状态

**接口**：`/api/sap/CE_PRODUCTIONORDER_0001/ProductionOrder?id=xxx&expand=_Component,_Operation`

---

### 2.6 视图切换 (P1)

**需求描述**：列表页支持卡片/表格两种视图切换。

**功能点**：
- 顶部切换按钮（图标切换）
- 默认卡片视图（移动端优先）
- 切换状态记录到本地存储

---

### 2.7 交货单详情页 (P2)

**需求描述**：从列表或销售订单详情点击交货单号进入详情页。

**功能点**：
- 头部信息：交货单号、类型、客户、日期、状态
- 行项目列表：物料、数量、单位
- 关联销售订单信息

---

### 2.8 开票单据详情页 (P2)

**需求描述**：从列表或销售订单详情点击开票单号进入详情页。

**功能点**：
- 头部信息：开票单号、类型、客户、日期、金额、状态
- 行项目列表：物料、金额、数量
- 关联销售订单/交货单信息

---

## 3. 非功能需求

### 3.1 性能
- 列表页首屏加载 < 2s
- 详情页加载 < 3s
- 搜索响应 < 1s

### 3.2 兼容性
- 微信版本 ≥ 8.0
- iOS 14+ / Android 8+

### 3.3 网络
- 弱网环境下降级提示
- API 超时 15s
- 请求失败自动重试 1 次

## 6. 实施优先级

| 优先级 | 功能 | 理由 |
|--------|------|------|
| P0 | 登录认证 | 安全基础，所有功能前提 |
| P0 | 首页 Dashboard | 小程序门面，用户第一印象 |

| P1 | 视图切换 | 移动端体验优化 |

| P2 | 开票单据详情页 | 从销售订单详情可跳转 |
